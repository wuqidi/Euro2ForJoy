# Euro2ForJoy
游戏手柄映射

## 功能

>* 识别joystick按键位，并进行键盘映射
>* 映射文件:resources/keymapping.yml

## 瞎JB忙
* 有时间就写点

##使用方式
```
下载soft包下所有文件，然后随便存入一个独立的文件夹中。
例如 G:/mygames/
dll:识别硬件的运行库
img:图片 没毒
keymapping.yml：摇杆按键的映射规则文件
```
